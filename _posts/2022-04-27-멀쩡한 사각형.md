```python
w = 8
```

```python
def solution(w,h):

    m = max(w,h)
    n = min(w,h)

    while True:
        tmp = m % n
        if tmp == 0:
            gcd = n
            break
        m, n = max(n,tmp), min(n,tmp)

    return w*h - (w+h-gcd)
```


